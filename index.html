Nice Toopltip

lightweight jQuery plugin

Welcome! This small $ plugin was developed as simple and elegant way to add tooltips to your text.

You can have one instance of tooltip – well, too much isn't that good, huh? This is absolutely simple solution – in most, you manage position and HTML of tooltip (but anyway, that's enough).

Settings

Mostly, you need only one thing – override default css. You can do it by passing css object into options.
